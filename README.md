## Samenvatting
In het voorjaar 2025 is er sterke behoefte om op dezelfde manier als er een OKE gedefinieerd is voor examens in het MBO, een standaard te ontwikkelen op dezelfde principes voor de BPV (beroeps praktijk vorming) in het MBO in nederland.
Dit zou moeten starten met het refereren van de relevante MORA processen, het definieeren van de flows tussen aplicatie componenten en die uitdrukken als een API gebaseerd op de OOAPI. (open onderwijs API)

Een groep van scholen, software leveranciers heeft beloten dit als OKx best practise koppeling op te leveren. Hiermee kan elke partij in het MBO deelnemen in de BPV gerelateerde processen, ongeacht welke SIS, SVS, ELO, BPV matching of begeleiding applicaties er bij een onderwijs instelling in gebruik zijn.

Status Mei 2025: Start
